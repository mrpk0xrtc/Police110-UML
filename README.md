# Police 110 UML

Modeling Police Oraganization using UML

## Requirements

The Following is requirements to compile and generate pdf file:

  - XeTeX
    - XePersian
    - bidi
  - InkScape
  - ImageMagick
  - PlantUML

A `logo.png` file is also needed for compilation.

## Build

To generate final pdf file run following commands:

```
make gen_logo
make gen_diagrams
make build
```
