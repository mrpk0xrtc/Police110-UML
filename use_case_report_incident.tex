\subsection{\lr{Use Case - Report Incident}}

\hspace{1cm}
فرآیند گزارش یک حادثه که از طریق تماس شهرواندان به دلیل حوادث اضطراری یا مشکوک, از طریق شماره تماس تلفنی یا وبگاه اینترنتی آغاز شده و مراحل پاسخ به حادثه شروع می شود.

\hspace{1cm}

\begin{enumerate}
    \item شهروند با پلیس تماس می گیرد (110/اینترنت)
    \item گیرنده ی تماس, پاسخ می دهد. ابتدا موقعیت حادثه را جمع آوری می کند.
    \item تشخیص نوع و اولویت حادثه
    \item جمع آوری اطلاعات حیاتی
    \item اعزام نیرو های مناسب
    \item تهیه دستورالعمل های مناسب در صورت نیاز
    \item رصد عملیات تا خاتمه
\end{enumerate}
